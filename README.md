# README


### 后端目录在server文件夹下，node server.js可运行


### install yarn

see https://classic.yarnpkg.com/zh-Hans/docs/install/
### install dependencies
run `yarn install`
### run locally:
run `yarn start`
### access:
http://localhost:3030

### production build
run `yarn build`
### install serve
`yarn global add serve`
### production run
`serve -s build`
